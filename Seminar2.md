# Seminar number 2

### Branch creating

* *git branch <name>* - создать ветку в git

* *git branch* - отображает список существующих веток

### Merging

* *git merge <branch name>* - слияние ткущей ветки с branch name

* *git branch -d <branch name>* - удалить ветку branch name
